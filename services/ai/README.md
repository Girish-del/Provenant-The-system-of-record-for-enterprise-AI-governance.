# aegis-ai (Python FastAPI)

AI service: risk classification, impact-assessment drafting, control-mapping
suggestions, and RAG Q&A over framework texts (pgvector). Talks to the Anthropic
Claude API. Internal-only; never internet-reachable. All output is advisory, labeled,
sourced, provenance-logged, and never auto-applied (see `docs/05-SECURITY-AND-THREAT-MODEL.md`).

Scaffolded in component M9. See `BUILD-LOG.md`.
